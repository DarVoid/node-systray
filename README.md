# node-systray

> SysTray library for nodejs using [systray-portable](https://github.com/zaaack/systray-portable) (a portable version of [the go systray library](https://github.com/getlantern/systray)).


## Install
```sh
npm i systray
# or
yarn add systray
```

## Usage

```js
import SysTray from 'systray'

```
